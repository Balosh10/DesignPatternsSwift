# Strategy

Patrón de Comportamiento — Strategy

Descripción

Ejemplo didáctico del patrón Strategy con un playground demostrativo.

---

## Teoría

Strategy define una familia de algoritmos intercambiables y los encapsula, permitiendo que el algoritmo varíe independientemente de los clientes que lo usan. Favorece el principio de abierto/cerrado y reduce condicionales.

**Cuándo usarlo:** cuando necesites seleccionar dinámicamente una política o algoritmo en tiempo de ejecución.
