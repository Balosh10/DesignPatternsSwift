# Delegate

Patrón de Comportamiento — Delegate

Descripción

Ejemplo didáctico del patrón Delegate (muy usado en iOS).

---

## Teoría

Delegate es un patrón de comportamiento que permite a un objeto delegar responsabilidades a otro, estableciendo una relación uno-a-uno o uno-a-muchos mediante protocolos. Es ampliamente usado en iOS para manejar callbacks y eventos sin acoplar las clases.

**Cuándo usarlo:** cuando quieras separar la lógica de respuesta a eventos del originador del evento, manteniendo bajo acoplamiento.
