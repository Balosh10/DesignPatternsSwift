# Command

Patrón de Comportamiento — Command

Descripción

Ejemplo didáctico del patrón Command con un playground demostrativo.

---

## Teoría

Command encapsula una solicitud como un objeto, permitiendo parametrizar clientes con diferentes solicitudes, cola de operaciones o deshacer/rehacer. Separa el emisor de la acción del objeto que la ejecuta.

**Cuándo usarlo:** cuando necesites desacoplar emisores y receptores, implementar colas de ejecución o soportar operaciones reversibles.
