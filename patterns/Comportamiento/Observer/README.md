# Observer

Patrón de Comportamiento — Observer

Descripción

Ejemplo didáctico del patrón Observer con un playground demostrativo.

---

## Teoría

Observer define una dependencia uno-a-muchos entre objetos de modo que cuando uno cambia, sus dependientes son notificados automáticamente. Es útil para diseñar sistemas reactivos y desacoplados.

**Cuándo usarlo:** cuando múltiples objetos deben reaccionar a cambios en otro objeto sin acoplarse fuertemente.
