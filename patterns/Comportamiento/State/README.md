# State

Patrón de Comportamiento — State

Descripción

Ejemplo didáctico del patrón State con un playground demostrativo.

---

## Teoría

State permite a un objeto alterar su comportamiento cuando su estado interno cambia, pareciendo cambiar su clase. Evita grandes condicionales y organiza lógica por estados.

**Cuándo usarlo:** cuando un objeto tiene múltiples estados con comportamiento distinto y quieres encapsular la lógica de cada estado.
