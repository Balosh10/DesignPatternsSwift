# Adapter

Patrón Estructural — Adapter

Descripción

Ejemplo didáctico del patrón Adapter con un playground demostrativo.

---

## Teoría

Adapter permite que interfaces incompatibles colaboren convirtiendo la interfaz de una clase en otra que el cliente espera. Es útil para integrar librerías o APIs existentes sin modificar su código.

**Cuándo usarlo:** cuando necesites adaptar código legado o terceros a la interfaz que tu sistema requiere.
