# Composite

Patrón Estructural — Composite

Descripción

Ejemplo didáctico del patrón Composite con un playground demostrativo.

---

## Teoría

Composite permite tratar objetos individuales y compuestos de manera uniforme, formando estructuras de árbol. Facilita operaciones recursivas sobre componentes y reduce la complejidad del cliente al manipular la estructura.

**Cuándo usarlo:** cuando tengas jerarquías de objetos y necesites operaciones que funcionen tanto en elementos simples como en agregados.
