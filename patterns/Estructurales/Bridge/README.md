# Bridge

Patrón Estructural — Bridge

Descripción

Ejemplo didáctico del patrón Bridge con un playground demostrativo.

---

## Teoría

Bridge separa una abstracción de su implementación, permitiendo que ambas varíen independientemente. Evita combinaciones explosivas de subclases cuando hay múltiples dimensiones de variación.

**Cuándo usarlo:** cuando quieras desacoplar la interfaz de las implementaciones y soportar extensibilidad en ambas direcciones.
