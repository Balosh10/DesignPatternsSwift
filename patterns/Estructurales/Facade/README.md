# Facade

Patrón Estructural — Facade

Descripción

Ejemplo didáctico del patrón Facade con un playground demostrativo.

---

## Teoría

Facade proporciona una interfaz simplificada a un conjunto de interfaces en un subsistema, haciéndolo más fácil de usar. Es útil para ocultar complejidad y definir puntos de entrada claros para funcionalidades complejas.

**Cuándo usarlo:** cuando quieras reducir la dependencia de clientes sobre complejos subsistemas o agrupar varias operaciones en APIs simples.
