# Factory Method

Patrón Creacional — Factory Method

Descripción

Implementación didáctica del Factory Method. Añade ejemplos y un playground demostrativo.

Ubicación del ejemplo

- Playground: `FactoryMethod.playground`

Cómo usar

1. Abre el playground en Xcode.
2. Ejecuta el playground para ver ejemplos.

---

## Teoría

El Factory Method define una interfaz para crear objetos, pero permite que las subclases decidan qué clase instanciar. Promueve la desacoplación entre el código que usa objetos y las clases concretas que los implementan.

**Cuándo usarlo:** cuando necesitas delegar la creación de instancias a subclases o conservar una interfaz estable mientras varían las implementaciones concretas.
