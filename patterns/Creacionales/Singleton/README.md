# Singleton

Patrón Creacional — Singleton

Descripción

Ejemplo educativo de `Singleton` con uso seguro en Swift.

---

## Teoría

El Singleton garantiza que una clase tenga una única instancia y proporciona un punto de acceso global a ella. En Swift se debe usar con precaución, evitando abuso que lleve a estado global difícil de testear.

**Cuándo usarlo:** cuando realmente se necesita un único objeto compartido (por ejemplo, un gestor de configuración o cache global), y siempre documentando su razón de ser.
