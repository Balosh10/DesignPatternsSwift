# Abstract Factory

Patrón Creacional — Abstract Factory

Descripción

Implementación didáctica de Abstract Factory. Incluye un playground con ejemplos.

---

## Teoría

Abstract Factory proporciona una interfaz para crear familias de objetos relacionados sin especificar sus clases concretas. Es útil cuando un sistema debe ser independiente de cómo se crean sus productos y cuando hay familias de productos que deben mantenerse consistentes.

**Cuándo usarlo:** cuando necesitas garantizar compatibilidad entre productos de una misma familia y permitir distintas implementaciones intercambiables.
