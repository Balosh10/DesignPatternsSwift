# Creacionales

Patrones creacionales: controlan la creación de objetos.

Orden recomendado para estudiar este grupo:

1. Builder
2. Factory Method
3. Abstract Factory
4. Prototype
5. Singleton

Este orden facilita comprender primero la construcción paso a paso de objetos y luego pasar a soluciones más flexibles de creación.

Cada patrón tiene su carpeta con ejemplos y un `README.md` propio.
