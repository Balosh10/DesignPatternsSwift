# Uso del proyecto

Este proyecto contiene implementaciones educativas de patrones de diseño en Swift.

Requisitos mínimos

- Xcode 12 o superior
- Swift 5

Abrir y ejecutar

1. Abre Xcode.
2. En el Finder, navega a la carpeta del repositorio y abre `Builder.playground` o `PD-Builder.playground`.
3. Ejecuta el playground para ver los ejemplos y el output.

Estructura recomendada

- `patterns/` — contiene carpetas por categoría y patrón. Cada patrón debe incluir un `README.md` explicando el uso y un archivo de demostración (playground o swift file).

Buenas prácticas

- Añade tests o pequeños snippets que demuestren el uso del patrón.
- Documenta las decisiones arquitectónicas en el `README.md` del patrón.
