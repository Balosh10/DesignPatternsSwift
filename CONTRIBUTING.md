# Contribuir

Gracias por querer contribuir a DesignPatternsSwift.

Cómo contribuir

1. Haz fork del repositorio.
2. Crea una rama con un nombre descriptivo: `feature/<patron>-<descripcion>`.
3. Añade tu implementación bajo `patterns/<Categoria>/<Patron>/`.
4. Incluye un `README.md` para el patrón, ejemplos y, si es posible, tests.
5. Abre un Pull Request describiendo los cambios y el motivo.

Estilo de código

- Sigue las guías de estilo de Swift.
- Mantén funciones y tipos con responsabilidad única.
